# NaturalLanguageProcessingTutorials

This project provides resources for the understanding and implementation of Natural Language Processing (NLP) techniques. These are available in the folder *notebooks*, listed as follows:

- 0. NLP main tools

Main available python tools to use for NLP tasks

- 1. Pre-Processing

Pre-processing techniques in NLP and how to use the available resources for this purpose.

- 2. Feature Extraction

How to perform Feature extraction with the resources provided, namely, how to convert text to a quantifiable measure.

- 3. Text Processing Methods

Three main methods for text processing are provided: LDA, NMF and LSI, and applied to the context of Text Modelling.

- 4. Applications to Text Mining

How to apply the provided resources to specific text mining exercises, namely for Biomedical text mining.
